# Text Analysis Tool

Инструмент для анализа текста с возможностью извлечения намерений и визуализации результатов.

## Возможности

- Анализ текстовых файлов
- Извлечение намерений с использованием примеров
- Подсчет частоты слов
- Создание визуализаций:
  - Облако слов
  - Столбчатые диаграммы
  - Тепловые карты
  - Распределение намерений
- Генерация HTML-отчетов
- Консольный интерфейс для быстрого анализа

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/text-analysis-tool.git
cd text-analysis-tool
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

### Консольный интерфейс

Для быстрого анализа используйте `usage_example.py`:
```bash
python big/usage_example.py
```

### Расширенный анализ

Для более детальной настройки используйте `_main.py`:
```bash
python big/_main.py --config config.yml --files "path/to/your/file.txt"
```

## Структура проекта

```
text-analysis-tool/
├── big/
│   ├── _main.py              # Основной скрипт для расширенного анализа
│   ├── usage_example.py      # Консольный интерфейс для быстрого анализа
│   ├── text_analyzer.py      # Класс для анализа текста
│   ├── visualization.py      # Класс для визуализации результатов
│   ├── multilingual.py       # Поддержка мультиязычности
│   └── config.yml            # Пример конфигурационного файла
├── requirements.txt          # Зависимости проекта
└── README.md                 # Документация
```

## Конфигурация

Настройки анализа можно задать в файле `config.yml`:

```yaml
settings:
  chunk_size: 1000
  language: auto
  num_workers: 4
  similarity_threshold: 0.6
  top_n_words: 50
  output_dir: results

intent_examples:
  мыслеформа_потребительства:
    - "Чем больше покупок, тем лучше"
    - "Новые приобретения делают человека счастливее"
    - "Качество жизни определяется количеством вещей"
  мыслеформа_развития:
    - "Постоянное обучение - ключ к успеху"
    - "Саморазвитие важнее материальных ценностей"
    - "Инвестиции в знания приносят лучшие дивиденды"
```
